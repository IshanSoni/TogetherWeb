## Getting Started
<br>
1. Clone this repo - `git clone https://github.com/ayushyamitabh/musicroom`
2. Go to the web directory - `cd musicroom-web`
3. Install dependencies - `npm install`
4. To start the local server [on localhost:3000] - `npm start`
